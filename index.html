CoreData viðbætur

CoreData Desktop 4.0.1

Útgáfa 4.0.1 af CoreData Desktop bætir við stuðningi við vafra sem ekki styðja lengur NPAPI(4). Einning er stuðningi við eldri .NET umhverfi (eldri en .NET 4.5) hætt. Þessi útgáfa gerir ráð fyrir útgáfu 2.11+ af CoreData, sem stendur til að uppfæra viðskiptavini í um miðjan september 2015.

  • CoreData Desktop fyrir Windows 64b, .NET 4.5
  • CoreData Desktop fyrir Windows x86, .NET 4.5

CoreData Desktop 3.2.0

CoreData Desktop er fáanlegt fyrir Windows stýrikerfi. Mismunandi útgáfur eru til fyrir 64 bita örgjörva annars vegar og hins vegar x86(1). Einnig skiptir máli hvort .NET 4.0 eða nýrra sé uppsett eða eldri útgáfa (.NET 3.5)(2).

  • CoreData Desktop fyrir Windows 64b, .NET 3.5
  • CoreData Desktop fyrir Windows 64b, .NET 4.0+
  • CoreData Desktop fyrir Windows x86, .NET 3.5
  • CoreData Desktop fyrir Windows x86, .NET 4.0+

Outlook Addin 1.5.0

Outlook viðbótin er fáanleg fyrir Windows stýrikerfi. Tvær útgáfur eru til, annars vegar fyrir .NET 3.5 og hins vegar .NET 4.0 og nýrri(2). Ekki skiptir máli hvort um 64b eða x86 örgjörva er að ræða.

Outlook viðbótin styður Office 2010 og Office 2013.

  • Outlook Addin fyrir .NET 3.5
  • Outlook Addin fyrir .NET 4.0+
